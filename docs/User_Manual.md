## G. APPENDIX

### 1. Keyboard Shortcuts:
   **Podcast Studio:**
   - Space: Play/Pause
   - R: Start/Stop recording
   - M: Mute selected channel
   - S: Solo selected channel
   - Ctrl+S: Save project
   - Ctrl+E: Export

   **Video Editor:**
   - I: Set in point
   - O: Set out point
